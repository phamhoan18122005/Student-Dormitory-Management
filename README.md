# Student Dormitory Management Database

## Overview

This project is a MySQL database designed for managing a student dormitory.

The database supports managing students, rooms, contracts, registrations, facilities, employees and billing information.

---

## Technologies

- MySQL
- SQL
- MySQL Workbench

---

## Features

- Student Management
- Room Management
- Dormitory Building Management
- Contract Management
- Registration Management
- Billing Management
- Employee Management
